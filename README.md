# supply_automation
### Опт
- **Оптовые заказы FNow с увеличением до суммы** - создание оптовых заказов с возможностью увеличения до заданной суммы
- **Заказы Китай - NPS** - обработка заказов из Китая с учетом NPS (Net Promoter Score)

### Аналитика
- **Максимальный запас** - расчет максимального запаса товаров с учетом сезонности, прогнозов и текущих акций
- **Блокировки** - инструменты для расчета различных типов блокировок (EB, PB, BN, RB)
- **Наличие** - анализ наличия товаров (розничные продажи, корпоративные продажи, РФ)
- **Неликвиды** - выявление и анализ неликвидных товаров с настраиваемыми порогами
- **Наличие по форматам** - анализ наличия товаров по различным форматам магазинов
- **NPS** - анализ Net Promoter Score
- **Минимальная партия** - расчет минимальных партий заказов с настраиваемыми параметрами
- **SKU в поставках** - анализ ассортимента в поставках

### Розница
- **Допзаказ ПРК** - дополнительные заказы для ПРК (Промо-розничная кампания)
- **График** - инструменты для анализа графиков транспортировки

### Цены
- **Наценка** - расчет наценок на товары
